Each database is pre-processed independently. Consequently, the different Rcodes stored in this folder must be run. The generated codes use functions located in the [scripts](./scripts/) folder. The 4.1 version of R was used.

These Rcodes automatically load the datasets whenever possible. However, programming access is sometimes blocked by the authors, so some files must be downloaded manually.
